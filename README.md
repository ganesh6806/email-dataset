# email-dataset
Unstructured to Structured email dataset

Tools Required
	Node/NPM
	CORS
	ELASTIC SEARCH
	ANGULAR CLI
	
Steps to execution

Prerequisites
Node > 4.0
CORS
ELASTIC SEARCH
Download and extract public Enron email dataset here. place the maildir into below folder:
root/DATA/maildir

How to run
Clone the repository, run

npm install
npm install elasticsearch
npm install cors
node parse.js
node import.js
node server.js

Check the application in http://127.0.0.1:3000/
